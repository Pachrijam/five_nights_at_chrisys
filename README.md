# 🎮 Five Nights at Chrisy's

A Unity horror game demo inspired by survival-style night mechanics. Stay alive, get your goon in, and make it through the night at Chrisy’s... if you can.

---

## 🧠 About the Game

**Five Nights at Chrisy's** is a small demo project built in Unity where the player must survive multiple nights against unknown threats. The gameplay focuses on tension, timing, and decision-making.

---

## ⚙️ Built With

- Unity (Game Engine)
- C#
- Visual Studio / VS Code

---

## 🎯 Features

- 🕶️ First-person survival gameplay  
- ⏱️ Night-based progression system  
- 🔊 Atmospheric sound design  
- 🚪 Basic interaction mechanics (lights, cameras, etc.)  
- 👀 Enemy Level Progression
